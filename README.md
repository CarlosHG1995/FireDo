# FireDo
Un los servicios cloud  conocidos en la actualidad, Firebase, en concreto con 
tres de los servicios más comúnmente utilizados: Authentication para la gestión de usuarios, 
Realtime Database como base de datos entiempo real
y Storage como sistema de almacenamiento de ficheros en cloud.

Para añadir las dependencias necesarias se realiza a través de Cocoapods.

Funcionamiento de la app. Un usuario
autenticado podrá tanto crear como unirse a un grupo de trabajo ya existente, así como crear
tareas en cada uno de los grupos a los que se haya unido. Cuando un usuario crea una tarea
dentro de un grupo, esa nueva tarea será accesible en tiempo real para el resto de usuarios.
